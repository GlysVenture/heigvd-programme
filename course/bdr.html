BDR | Bases de données relationnelles

Crédits ECTS: 5.0

L'objectif de cette unité est de maîtriser les éléments essentiels pour la conception, la mise en œuvre et l'utilisation des bases de données relationnelles, plus précisement : concevoir les schémas conceptuels et relationnels de base de données avec une démarche d'ingénieur ; comprendre et utiliser le modèle relationnel, les contraintes d'intégrité et l'algèbre relationnelle ; implémenter une base de données sur un systéme de gestion de base de données (SGBD) relationnelles et utiliser le langage SQL pour la définition, la manipulation et le contrôle des données ; décrire les formes normales et les appliquer pour vérifier la qualité d'une base de données ; utiliser les bases de données au travers un programme Java via l'API JDBC.
