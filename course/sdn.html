SDN | Software-Defined Networking

Crédits ECTS: 4.0

L'objectif de cette unité est de comprendre la transformation actuelle des réseaux qui est amenée par deux nouveaux concepts : les réseaux pilotés par logiciel (SDN - Software Defined Networking) ; la virtualisation des fonctions réseau (NFV - Network Function Virtualization). Elle permet de se familiariser avec les Clouds dédiés aux fonctions réseau (Telco Cloud), la virtualisation, les switches virtuels, les réseaux "overlay" et des APIs pour automatiser la configuration, le déploiement et le monitoring de solutions basées sur SDN/NFV.
